Overview

A full-stack web application that extracts text from PDF and image files and provides engagement-based content analysis for social media posts.


Tech Stack
Frontend: Next.js, TypeScript, Tailwind CSS
Backend: Node.js, Express
OCR: Tesseract.js
PDF Parsing: pdf-parse
Deployment: Vercel (Frontend), Render (Backend)


Features

Drag-and-drop file upload
PDF and image text extraction
Word count analysis
Hashtag detection
Call-to-action detection
Engagement improvement suggestions
Loading states and error handling

Live Demo
Frontend: https://social-media-analyzer-one.vercel.app/


How It Works
User uploads a file.

Backend extracts text (PDF or OCR).

Frontend analyzes engagement metrics.

Suggestions are generated dynamically.
